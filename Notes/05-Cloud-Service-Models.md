# ☁️ Chapter 5 - Cloud Service Models

## What are Cloud Service Models?

Cloud Service Models define **what the cloud provider manages** and **what the customer manages**.

There are **3 main cloud service models:**

1. 🖥️ IaaS (Infrastructure as a Service)
2. ⚙️ PaaS (Platform as a Service)
3. 💻 SaaS (Software as a Service)

---

# 🖥️ 1. IaaS (Infrastructure as a Service)

## Definition

In **IaaS**, the cloud provider gives you the infrastructure such as **servers, storage, and networking**, while you install and manage the operating system, applications, and data.

You have the **most control** over your environment.

### Real-Life Example

Imagine renting an **empty apartment**.

The owner provides:

- Building
- Electricity
- Water

You bring:

- Furniture
- TV
- Bed
- Kitchen items

Similarly, the cloud provider gives you the infrastructure, while you install and manage everything else.

### Examples

- Virtual Machines
- Virtual Storage
- Virtual Networking

### Customer Manages

- Operating System
- Applications
- Data
- Security inside the Operating System

### Cloud Provider Manages

- Physical Servers
- Storage Hardware
- Networking Hardware
- Data Centers

---

# ⚙️ 2. PaaS (Platform as a Service)

## Definition

In **PaaS**, the cloud provider manages both the infrastructure and the operating system.

You only focus on developing and deploying your application.

### Real-Life Example

Imagine renting a **fully furnished apartment**.

Everything is ready.

You simply move in and live there.

Similarly, developers only upload their application code.

### Examples

- Application Hosting Platforms
- Web Application Platforms

### Customer Manages

- Application Code
- Data

### Cloud Provider Manages

- Servers
- Storage
- Networking
- Operating System
- Runtime Environment

---

# 💻 3. SaaS (Software as a Service)

## Definition

In **SaaS**, everything is managed by the service provider.

You simply access and use the software through a web browser or mobile application.

No installation.

No maintenance.

### Real-Life Example

Using **Gmail**.

You don't install mail servers.

You simply sign in and start using it.

### Examples

- Gmail
- Google Drive
- Microsoft 365
- Zoom
- Dropbox

### Customer Manages

Almost nothing.

Just use the software.

### Service Provider Manages

Everything.

---

# Service Model Comparison

| IaaS | PaaS | SaaS |
|------|------|------|
| Rent Infrastructure | Rent Platform | Use Software |
| Most Control | Medium Control | Least Control |
| You manage OS & Applications | You manage only Application & Data | Provider manages everything |

---

|---------------|------------------|------------------|
