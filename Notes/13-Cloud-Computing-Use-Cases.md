# ☁️ Chapter 13 - Cloud Computing Use Cases

## What are Cloud Computing Use Cases?

A **use case** describes **how and where Cloud Computing is used in real life.**

Today, almost every industry uses Cloud Computing to improve performance, reduce costs, and provide better services.

---

# 1. Banking 🏦

Banks use Cloud Computing to:

- Store customer data
- Process online transactions
- Support mobile banking
- Provide internet banking services
- Detect fraudulent activities

### Example

When you use:

- Google Pay
- PhonePe
- SBI YONO

Cloud Computing helps process your transactions quickly and securely.

---

# 2. Healthcare 🏥

Hospitals use Cloud Computing to:

- Store patient records
- Manage online appointments
- Store medical reports
- Provide video consultations

### Example

A doctor can access your medical records from any hospital branch.

---

# 3. Education 🎓

Schools and colleges use Cloud Computing for:

- Online classes
- Digital notes
- Online exams
- Student records

### Examples

- Google Classroom
- Microsoft Teams
- Zoom

---

# 4. E-Commerce 🛒

Shopping websites use Cloud Computing to:

- Store product details
- Process customer orders
- Handle online payments
- Manage customer accounts

### Examples

- Amazon
- Flipkart
- Myntra

During shopping festivals and sales, Cloud Computing helps these websites handle millions of users.

---

# 5. OTT Platforms 🎬

OTT platforms use Cloud Computing to:

- Store movies and TV shows
- Stream videos
- Recommend content based on user preferences

### Examples

- Netflix
- Disney+ Hotstar
- Amazon Prime Video

---

# 6. Social Media 📱

Social media platforms use Cloud Computing to:

- Store photos and videos
- Send messages
- Deliver notifications

### Examples

- Instagram
- Facebook
- WhatsApp

Millions of photos and videos are uploaded and stored every day using cloud infrastructure.

---

# 7. Gaming 🎮

Online games use Cloud Computing to:

- Save player progress
- Connect players around the world
- Host multiplayer matches

### Examples

- PUBG
- Free Fire
- Fortnite

---

# 8. Business & Companies 💼

Companies use Cloud Computing for:

- Employee data
- Email services
- File sharing
- Video meetings
- Customer management

### Example

A company stores its documents in the cloud so employees can securely access them from different office locations.

---

# Advantages of Cloud Computing in Real Life

- ✅ Access data from anywhere
- ✅ Lower infrastructure cost
- ✅ Fast performance
- ✅ Easy collaboration
- ✅ Better backup and recovery
- ✅ High availability
