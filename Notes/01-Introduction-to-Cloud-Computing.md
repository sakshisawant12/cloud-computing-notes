**☁️ Chapter 1 - Introduction to Cloud Computing**

What is Cloud Computing?
Definition
**Cloud Computing means using computing services (servers, storage, database, networking, software) over the Internet instead of buying and maintaining them yourself.**
Simple Example
Imagine you want to watch a movie.
❌ Old Way
•	Buy a DVD 
•	Buy a DVD player 
•	Store everything 
✅ Cloud Way
•	Open Netflix 
•	Watch instantly 
Cloud computing works the same way.
Instead of buying servers, you rent them whenever you need them.
________________________________________

 
 Why do we need Cloud Computing?
Earlier, companies had to:
•	Buy servers 💰 
•	Buy storage 💰 
•	Hire engineers 👨‍💻 
•	Maintain everything 😓 
It was expensive and took a lot of time.
Now with cloud:
•	Rent resources 
•	Pay only for what you use 
•	No maintenance 
•	Ready in minutes

⭐ Characteristics of Cloud Computing
Cloud Computing has 5 main characteristics.
________________________________________
1. On-Demand Self-Service
Users can create or remove cloud resources whenever they need them, without contacting the cloud provider.
Example
A developer creates a virtual server in a few minutes using a cloud portal.
________________________________________
2. Broad Network Access
Cloud services can be accessed from anywhere using the Internet.
Example
You can access Google Drive from your phone, laptop, or tablet.
________________________________________
3. Resource Pooling
The cloud provider shares physical resources (servers, storage, etc.) among many customers while keeping each customer's data separate and secure.
Example
Many users use the same cloud infrastructure, but their data remains isolated.
________________________________________
4. Rapid Elasticity
Cloud resources can increase or decrease quickly based on demand.
Example
During a festival sale, an e-commerce website automatically gets more resources to handle extra traffic.
________________________________________
5. Measured Service
Customers pay only for the resources they actually use.
Example
If you use a cloud server for 5 hours, you pay only for those 5 hours.

⭐ Cloud Migration
What is Cloud Migration?
Cloud Migration is the process of moving applications, data, databases, or workloads from an on-premises environment to the cloud.
Simple Definition
Cloud Migration means moving your IT resources from your own servers to the cloud.
________________________________________
Why do companies migrate to the cloud?
•	✅ Lower cost 
•	✅ Better scalability 
•	✅ Easy management 
•	✅ High availability 
•	✅ Better backup and disaster recovery 
________________________________________
Example
A company hosts its website on its own server.
Later, it moves the website and database to a cloud platform.
This process is called Cloud Migration.
________________________________________
Real-Life Example
Imagine you're moving from an old house to a new apartment.
You pack all your belongings and shift them to the new place.
Similarly, in cloud migration, a company moves its applications and data from its own servers to the cloud.
