# ☁️ Chapter 2 - Evolution of Cloud Computing

## What is Evolution of Cloud Computing?

**Evolution** means **how cloud computing has changed and improved over time**.

Earlier, companies managed everything themselves. As technology improved, cloud computing became easier, faster, and more affordable.

---

## Stage 1: Traditional Computing (On-Premises)

In traditional computing, companies had to manage everything on their own.

### Companies had to:

- Buy servers
- Buy storage
- Build data centers
- Hire IT staff
- Maintain everything

### Example

Imagine you start a new company. Before launching your website, you need to buy:

- 10 Servers
- UPS
- Air Conditioners
- Network Cables

💰 This requires a huge investment.

---

## Stage 2: Virtualization

Instead of using **one server for one application**, virtualization allows **one physical server to run multiple Virtual Machines (VMs).**

### Example

Without Virtualization

```text
1 Server → 1 Application
```

With Virtualization

```text
        Physical Server
              │
      ┌───────┼───────┐
      │       │       │
     VM1     VM2     VM3
```

✅ Saves money

✅ Better hardware utilization

---

## Stage 3: Cloud Computing

Instead of buying physical servers, companies started renting computing resources from cloud providers.

### Example

Instead of:

- Buying servers
- Maintaining hardware

Companies simply:

- Launch a virtual server
- Pay only for what they use

---

## Timeline

| Year | Event |
|------|-------|
| 1960s | Idea of computing as a utility |
| 1990s | Virtualization became popular |
| 2002 | Cloud services started becoming commercially available |
| 2006 | Cloud computing became widely available |
| Today | Most companies use cloud computing |

---

## Why did Cloud Computing become popular?

- ✅ Lower cost
- ✅ No hardware maintenance
- ✅ Easy scalability
- ✅ Faster deployment
- ✅ Access from anywhere

---

## Quick Revision

| Stage | Description |
|--------|-------------|
| Traditional Computing | Companies managed their own hardware |
| Virtualization | One physical server runs multiple VMs |
| Cloud Computing | Resources are delivered over the Internet |

---

## Interview Question

### What is the evolution of cloud computing?

**Answer:**

Cloud computing evolved from traditional on-premises infrastructure to virtualization, and finally to cloud computing, where users access computing resources over the Internet instead of owning physical hardware.
